# Changelog

## [1.0.0] - 2026-06-11

### Added
- Tienda web minimalista con catálogo de 4 productos y carrito de compras
- Panel de administración con login protegido (admin/admin123)
- Personalización de colores en tiempo real vía CSS custom properties
- Chatbot flotante con respuestas automáticas por palabras clave
- Reenvío de mensajes a WhatsApp mediante enlace wa.me
- Animaciones GSAP al scroll y círculo decorativo que sigue al ratón
- Base de datos MySQL con tablas admins, settings y chat_messages
- Script de instalación automatizada (install.sh + setup.sql)
